Contexte & Rôle :
Agis en tant qu'Expert Développeur Web
Ta mission est de rédiger le code complet d'un site web d'apprentissage agrémenté de jeux pour gamifier l'apprentissage
Le site web peut être offline, il faut enregistré la progression en local storage
Les données des cours et des jeux peuvent être en fichier statiques json

le nom du site sera : Apprendre

Objectif de l'outil :
Pour un jeune public (enfant / adolescent) : créer différent support de formation sur les bases de l'informatique de façon triviale et amusante

il faut mémoriser le parcours de l'apprenant et ses scores en quiz ou en jeu
la première chose à faire et donc de demander le pseudo de l'apprenant, et de l'enregistrer, puis de présenter de manière séquentiellle les différents cours en restant abordable.

les cours qu'il faut avoir :

- fonctionnement d'un ordinateur
- fonctionnement d'une IA générative (entraînement, inférence...)
- fonctionnement d'internet au niveau mondial (avec les interconnexion, les câbles sous marin, etc...)
- fonctionnement d'un jeu vidéo en ligne (comment faire pour que tout le monde partage la même expérience...)

Le site doit demander le pseudo de l'utilisateur et lui proposer une image d'avatar parmi une liste d'une vingtaine de possibilité

Contraintes Techniques & Sécurité :

Le code doit être modulaire (séparation HTML, CSS, JS).

Gère les erreurs (ex: si l'utilisateur est hors ligne ou si l'API échoue).

Commente le code de manière concise en français.

Avec ou sans framework JS, le but est d'avoir un code concis et optimisé, qui tourne en local et en offline sur un navigateur.
